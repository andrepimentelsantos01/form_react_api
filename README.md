# Form React API

Teste de algumas funcionalidades do React usando **Vite**, **TailwindCSS**, **React Hook Form**, **React Query** e outras bibliotecas para formulários e visualização de dados.

---

## Pré-requisitos

Certifique-se de ter instalado no seu computador:

* [Node.js](https://nodejs.org/) (recomendado >= 18)
* [npm](https://www.npmjs.com/) ou [Yarn](https://yarnpkg.com/)

---

## Instalação

1. Clone o repositório:

```bash
git clone https://github.com/andrepimentelsantos01/form_react_api.git
cd form_react_api
```

2. Instale as dependências:

```bash
npm install
# ou
yarn install
```

---

## Scripts disponíveis

Você pode executar os seguintes comandos:

| Comando           | Descrição                                     |
| ----------------- | --------------------------------------------- |
| `npm run dev`     | Inicia o servidor de desenvolvimento (Vite)   |
| `npm run build`   | Cria a versão de produção do projeto          |
| `npm run preview` | Pré-visualiza a versão de produção localmente |

---

## Principais dependências

* **React** (`react`, `react-dom`) – Biblioteca principal do projeto
* **Vite** (`vite`) – Ferramenta de build e servidor de desenvolvimento
* **TailwindCSS** (`tailwindcss`, `autoprefixer`, `postcss`) – Framework CSS
* **React Hook Form** (`react-hook-form`, `@hookform/resolvers`) – Para formulários
* **React Query** (`@tanstack/react-query`) – Gerenciamento de estado assíncrono
* **Axios** (`axios`) – Chamadas HTTP
* **Framer Motion** (`framer-motion`) – Animações
* **Recharts** (`recharts`) – Gráficos
* **React Leaflet + Leaflet** (`react-leaflet`, `leaflet`) – Mapas
* **Zod** (`zod`) – Validação de dados
* **Lucide React** (`lucide-react`) – Ícones
